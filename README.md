## meetbowl frontend

### Stacks

- React

  - Styled Components
  - React Query
  - React Router Dom

- CI/CD
  - Github Actions

### Infrastructure

- Cloud Flare Pages

### Architecture

- components : 컴포넌트
- pages : 페이지 컴포넌트
- hooks : 리액트 훅스
- apis : 백엔드 API 호출 함수
- utils : 유틸 함수
- assets : 이미지, 폰트 등
- constants : 상수
